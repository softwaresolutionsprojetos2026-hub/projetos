# Documentação do Produto

Índice mínimo alinhado ao projeto0.

- `adr/`: decisões de arquitetura.
- `api/`: contratos e notas de integração.
- `infra/`: decisões operacionais e dependências externas.
- `post-mortems/`: análise de incidentes P0/P1.
- `sbom/`: plano de supply chain e inventário.
- `skills/`: convenções de uso de IA no repositório.
- `slo/`: objetivos de serviço.
- `templates/`: moldes reutilizáveis.
- `ONBOARDING.md`: primeiros passos.
- `RUNBOOKS.md`: operação e resposta a incidentes.
